📋 QUICK DEPLOY (Copy-Paste This):
─────────────────────────────────────────────────────────────────

flyctl apps destroy nh-tourism-db --yes && \
flyctl postgres create --name nh-tourism-db --region ewr --initial-cluster-size 1 && \
sleep 30 && \
flyctl postgres attach nh-tourism-db --app nh-tourism && \
flyctl secrets list --app nh-tourism && \
flyctl deploy

─────────────────────────────────────────────────────────────────

⚠️  IMPORTANT NOTES:
  • When prompted during postgres create, press ENTER for defaults
  • Wait for "Postgres cluster created" message before continuing
  • Check that DATABASE_URL appears in secrets list
  • If DATABASE_URL is missing, wait 60s and try attach again
